# 一个简单的用户登录系统

#### Description
该项目是一个简单的JavaWeb用户登录系统，其中包含注册信息，删除信息，修改信息，展示用户信息这四个功能，界面虽然简单，但是功能比较全面。数据库连接的是MySQL。比较好上手，适合新手练习JavaWeb项目。
